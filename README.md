# KaziLedger

KaziLedger is a **fast, reliable, and scalable payroll and worker management system** built using **FastAPI**, **SQLAlchemy**, and **PostgreSQL**. It allows you to manage workers, subcontractors, Mpesa transactions, payouts, and attendances efficiently.

---

## Features

- Manage **workers** and **subcontractors**
- Track **Mpesa transactions**
- Create **payout batches** and **payout items**
- Monitor **worker attendance**
- Database **migrations with Alembic**
- Secure configuration with **Pydantic v2** and `.env` files
- Compatible with **PostgreSQL**

---

## Tech Stack

- **Backend:** Python, FastAPI
- **Database:** PostgreSQL
- **ORM:** SQLAlchemy
- **Migrations:** Alembic
- **Settings management:** Pydantic v2
- **Version Control:** Git
